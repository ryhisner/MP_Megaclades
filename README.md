The "Megaclade" files here are spreadsheets that list the number of EPCI sequences in which each mutation pattern (MP) 
occurs in each "Megaclade," the expected number of MP occurrences of each Megaclade (based on its overall prevalence in EPCI
sequences), and the fold-increase of the actual number of sequences relative to the expected number.

Files that end in "Substitutions" deal with specific amino acid mutations, while those ending in "Postion_Only" deal with all amino acid mutations at a given site.

The EPCI_Megaclade_Counts file lists the overall counts for each Megaclade among all EPCI sequences.

The Megaclades are, broadly: pre-VOC, Alpha, Beta, Gamma, Delta, BA.1, BA.2, BA.4/5, XBB, and BA.2.86.
Details for the precise clades included in each Megaclade are listed in a table included in each spreadsheet.

The names and mutations of each MP are described in the paper. Two abbreviations used here are:
CWW = Cryptic Wastewater MP
BAL = Bronchoalveolar Lavage-associated MP
